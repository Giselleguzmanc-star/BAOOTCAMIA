📊 Proyecto de Análisis de Datos




🧠 Descripción

Este repositorio hace parte de mi proceso de formación en un bootcamp de Análisis de Datos.
El objetivo principal es aplicar conceptos fundamentales de análisis, limpieza y exploración de datos utilizando Python.

Este proyecto funciona como una prueba inicial de repositorio, donde se documenta el aprendizaje y la evolución en el manejo de datos.

🎯 Objetivos

Practicar el uso de Python para análisis de datos
Desarrollar habilidades en manipulación y limpieza de datos
Aplicar técnicas básicas de exploración (EDA)
Documentar el proceso de aprendizaje

⚙️ Tecnologías utilizadas

Python 3
Google Colab

📚 Librerías principales

pandas → manipulación de datos
numpy → operaciones numéricas
matplotlib / seaborn → visualización de datos

🚀 Cómo usar este proyecto

Clonar el repositorio:
git clone https://github.com/tu-usuario/tu-repositorio.git
Abrir el archivo en Google Colab o en tu entorno local
Ejecutar las celdas paso a paso para seguir el análisis
